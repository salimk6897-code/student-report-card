# 📊 Student Report Card - by NJR

So this was one of my very first Python projects and honestly 
I was pretty proud when it actually worked! 

I built this when I was learning Python basics — the idea was 
simple, enter marks for 3 subjects and the program figures out 
your grade and gives you a little motivational message.

Nothing fancy, but it was the moment I realized I could actually 
build something real with code!

## What it does

You enter a student's name and their marks for English, Science 
and Math. The program calculates the average, assigns a grade 
and gives a personalized message based on how they did.

## How to run it

Just make sure Python is installed and run:
python student_report_card.py

## Grading

- 90 and above → Grade A
- 80 to 89 → Grade B  
- 70 to 79 → Grade C
- Below 70 → Need Improvement

## Built with

Python 3.14 — functions, lists, loops, if/else, user input
